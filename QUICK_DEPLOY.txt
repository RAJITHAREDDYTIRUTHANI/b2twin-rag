# 🚀 Quick Deploy to Share with Friends!

## Recommended: Render.com (Free & Easy)

### Setup (2 minutes):

1. **Go to**: https://render.com
2. **Click**: "Get Started for Free"
3. **Sign in** with your GitHub account
4. **Click**: "New +" → "Web Service"
5. **Connect Repository**: 
   - Paste: `https://github.com/RAJITHAREDDYTIRUTHANI/b2twin-rag`
   - Click "Connect"
6. **Configure**:
   - Name: `biosphere2-rag`
   - Region: (closest to you)
   - Branch: `main`
   - Build Command: `pip install -r requirements_rag.txt`
   - Start Command: `gunicorn spectacular_rag_web_app:app`
7. **Add Environment Variable**:
   - Key: `ANTHROPIC_API_KEY`
   - Value: (your API key from https://console.anthropic.com/)
8. **Click**: "Create Web Service"
9. **Wait**: ~5 minutes for first deployment
10. **Done!** Your app is live! 🎉

### Your app URL:
```
https://biosphere2-rag.onrender.com
```

